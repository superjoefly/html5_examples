HTML Paragraphs define a paragraph.

This is an example of a paragraph.

Line Breaks are used to break a line without starting a new paragraph. They are empty and have no end tag. They can be used for things like poems:

Today is a good day.
Today was a good day.
Today will be a good day.
Today is always a good day.

The HTML pre Element defines preformated text, is displayed in a fixed-width font and it preserves both spaces and line breaks:

	   Today is a good day.
	   Today was a good day.
	   Today will be a good day.
	   Today is always a good day.
